
 # 📝 Simple Text Editor (Python Tkinter)

A minimalist, beginner-friendly **text editor** built with Python's built-in `tkinter` library.  
Perfect for students learning **GUI programming**, **file handling**, and **event-driven programming**.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📂 Open File | Load any `.txt` file |
| 💾 Save / Save As | Save your work to a file |
| ✂️ Cut / Copy / Paste | Standard clipboard operations |
| 🆕 New File | Clear the editor to start fresh |
| ❌ Exit | Close the app with confirmation |
| ↩️ Undo Support | Built-in undo/redo (Tkinter default) |
